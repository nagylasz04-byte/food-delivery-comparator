# food-delivery-comparator
# Ételfutár-appok árainak összehasonlítása – Webalkalmazás

**Szerző:** Nagy László  
**Egyetem:** Debreceni Egyetem, Informatikai Kar  
**Téma:** Szakdolgozat / Diplomamunka – Webalkalmazás fejlesztés  

---

## Leírás

Ez a projekt egy webalkalmazás fejlesztését tartalmazza, amely lehetővé teszi a felhasználók számára, hogy összehasonlítsák az ételkiszállító appok árait egy adott rendelésre.  
A rendszer figyelembe veszi az étel árát, a szállítási díjat és minden rejtett költséget, majd kiemeli a legolcsóbb rendelési lehetőséget.

**Frontend:** HTML, CSS, JavaScript  
**Backend:** Python (Django)  
**Adatlekérés:** Web scraping vagy nyilvános API-k Pythonban  
**Opcionális:** Java modulok adatfeldolgozásra
